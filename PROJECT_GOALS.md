## Java 面试知识卡片 — 项目目标沉淀

### 项目定位

面向 3 年经验 Java 后端开发者的面试备战工具，目标公司是中型互联网公司。核心解决一个痛点：八股文资料多而散，复习没有体系、没有反馈、不知道哪里薄弱。

把 155+ 道高频面试题 + 项目经验话术 + 个人定制建议整合进一个单文件 HTML，打开浏览器就能学，不依赖任何后端服务。

---

### 核心目标

**1. 知识体系化**

把 Java 面试的 8 大方向（Java 基础、Spring、MySQL、Redis、消息队列、Docker/K8s、项目经验、复习计划）整合到一个页面里，用侧边栏导航切换，告别在十几个网页和 PDF 之间来回跳。每道题都是知识卡片格式：问题 → 答案 → 追问，结构清晰，适合碎片时间复习。

**2. 复习可量化**

每张卡片有"已复习"勾选框，侧边栏实时显示每个主题的完成进度（如 Java 12/25）。让复习从"感觉看了很多"变成"确实完成了 60%"，有明确的进度感。

**3. 自测驱动学习**

看答案不算会，能说出来才算。自测模式隐藏答案，先看题思考，再揭晓并自评（熟练/模糊/不会）。评分持久化到本地，下次打开接着用。支持按掌握程度筛选，集中攻克薄弱项。

**4. 模拟真实面试**

面试不按主题出题。面试模拟从所有方向随机抽 8 题（每方向最多 2 题），有计时器，模拟面试官跨主题跳问的节奏。结束后有成绩总结和逐题回顾。

**5. 选择题速测**

30 道精选四选一选择题，覆盖所有方向。选完即时反馈对错+解析，每轮随机 20 题打乱顺序。适合通勤、排队等碎片时间快速刷一轮。

**6. 个人定制化**

不是所有人都需要一样的复习计划。个人定制模块根据用户的技术栈、薄弱环节、项目亮点和面试时间，生成：专属面试画像、25 道必背重点题、项目亮点深挖话术、面试陷阱题预警、1-2 周冲刺计划。

**7. 项目经验包装**

3 年经验最容易翻车的地方不是八股文，而是项目介绍说不清楚。项目经验模块提供 STAR 法则模板、CRUD 项目亮点挖掘策略、技术难点描述框架、合理 QPS/数据量参考、10 大技术亮点方向、避坑指南。还有针对用户真实项目的完整面试话术（3 分钟逐字稿 + 追问应对表）。

---

### 技术实现目标

**单文件架构**：所有 HTML/CSS/JS 和数据都在一个文件里，无外部依赖，双击就能用。方便分享、拷贝到 U 盘、离线使用。

**零后端依赖**：所有数据存在 JS 变量里，用户状态（复习进度、自测评分）用 localStorage 持久化。不需要服务器、不需要数据库。

**响应式布局**：用 CSS `clamp()` 做弹性适配，侧边栏宽度按视口 16% 缩放（200-280px），内容区边距按 4vw 缩放（16-64px）。三档断点覆盖桌面、平板、手机。

**交互体验**：卡片点击展开/收起、搜索实时过滤（含追问内容）、自测模式/面试模拟/选择题三种模式互不干扰、全屏 overlay 做沉浸式答题。

### 后续可迭代方向

- **暗色模式**：护眼，适合夜间复习
- **个人笔记**：每张卡片下加笔记输入框，记录自己的理解和项目经历
- **间隔复习提醒**：根据自测评分计算下次复习时间，艾宾浩斯遗忘曲线
- **薄弱项统计面板**：全局雷达图，一眼看到各方向掌握程度
- **键盘快捷键**：数字键切换主题、空格展开卡片、N 下一题
- **打印友好版**：导出 PDF 方便纸质复习
- **多人协作**：支持自定义添加题目，团队共享题库

### 项目价值总结

这个项目的核心价值不是"又多了一份八股文资料"，而是：

1. **把被动阅读变成主动测试** — 自测模式和选择题模式强制你先想答案再看，学习效果比纯阅读强 3 倍以上（心理学上的"测试效应"）。
2. **把通用资料变成个人方案** — 不是给你 155 道题让你自己挑，而是根据你的简历和薄弱环节告诉你"这 25 道题优先背"。
3. **把知识点变成面试话术** — 不只告诉你 HashMap 怎么实现，还告诉你面试官追问"容量为什么是 2 的幂"时怎么回答。
4. **把复习变成可追踪的过程** — 进度条、评分、模拟成绩，让你的每一分钟复习都有据可查。
